== Description ==

Karma Website Builder is a Wp plugin that uses pre-made blocks to create websites without any need for either design or code knowledge.

<strong>Fully Drag & Drop</strong>

This intuitive drag and drop editor makes it easy to add elements, edit images or even rearrange the page.Create more in less time.


<strong>Live Text Editor</strong>

It is unbelievably easy to edit your text content with our outstanding inline text editor. Click on the content, edit font family, fonts size, color, style! Simple! 


<strong>Live Spacing Editor</strong>

With this handy tool you can arrange elements and adjust spacing between them visually.
Insanely quick and no page refreshes. 


<strong>Incredible Elements</strong>

With our powerful set of elements, you can make beautiful and customized WordPress websites. Incredible amount of design combinations are possible by Drag & Drop, allowing you to be creative without having any design experience.


<strong>No Coding Skills</strong>

No need to have any experience at all in coding. Karma Builder is so feature-rich and easy to use that you can create any type of website in any category without any coding knowledge.


<strong>Responsive</strong>

Karma Builder is a fully responsive plugin. Smartphone, tablet, laptop & desktop have individual display modes.

<strong>Modular Content Blocks</strong>

We have designed tons of pre-built sections, elements, blocks and even the whole website template to make the design process as easy as possible.


<strong>Highly Customizable</strong>

For predictable users & freelancers we have created several professional assets like responsive tools & typography to cover their needs.


<strong>Easy Development</strong>

Using karma API feature, you can develop any add-ons by utilizing interactive functions.


<strong>Works On All Devices</strong>

Our mobile-ready templates ensure a seamless and consistent experience that adapts to multiple devices.

== Installation ==

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)

 = Installation Guide = 
 
1. Upload `karma-builder.zip` to the `/wp-content/plugins/` directory and extract it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
